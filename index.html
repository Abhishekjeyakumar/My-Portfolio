Flight Scheduling Optimization

Flight Schedule Optimizer with NLP Queries is a Streamlit-based dashboard that integrates offline Excel datasets and live AviationStack API for flight data. It cleans, normalizes, and enriches flight schedules while automatically handling missing or inconsistent values. Users can ask natural language queries (e.g., "average delay", "worst airline") to get instant insights. The app also provides interactive visualizations such as delay trends, busiest hours, and airline comparisons. It helps identify delays, optimize scheduling, and support data-driven decision-making for flight operations.
